PySpark NLP: From Sentiment Analysis to Machine Learning Pipeline

Nesse projeto eu faço a análise de sentimentos utilizando o PySpark para processar dados do portal IMDB. Faço o fluxo completo de um projeto de ciência de dados, começando pela limpeza de textos com expressões regulares e a remoção de termos irrelevantes. Nele consigo mostrar a importância de transformar palavras em vetores numéricos e como organizar esse fluxo de trabalho através de um Pipeline otimizado. 

Criei um modelo de *árvore de decisão* treinado para classificar opiniões como positivas ou negativas. 
Por fim, a eficácia do sistema foi comprovada por meio de testes de acurácia e da validação bem-sucedida de novas frases inseridas no modelo.
